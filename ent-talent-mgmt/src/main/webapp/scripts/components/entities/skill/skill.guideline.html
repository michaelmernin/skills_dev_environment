Guidelines for Adding Skills

  1. No Ability : As it sounds

  2. Basic Ability : Have basic understanding or training but no practical experience (no client work)
  3. Functional Ability : Practical experience, used with clients, Intermediate knowledge
  4. Senior Ability : Experienced, deep knowledge, multiple engagements/multiple project experiences
  5. Ninja Ability : Advanced skills, very senior, Can train, mentor, and demonstrate most if not all facets of competency in this area.
